# Ignite Lab Node 2022

## Description

A NestJS framework project. A Microservice of notifications.
Main goal was to learn basics of NestJS. Interesting stuff!

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
